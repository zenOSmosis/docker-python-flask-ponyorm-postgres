# Prototype Python, Flash, PonyORM, and Postgres stack

## Prerequisites

- Docker
- Docker Compose

## To Run / Launch

```
docker-compose up
```

## To Tear Down

Press ctrl-C to stop.  Also, you may wish to run:

```
docker-compose down
```

## Browser

### To view the Python output:

```
http://localhost:5000
```

### To access Adminer interface:

```
http://localhost:8080
```
